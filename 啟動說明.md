# 🧦 AI 電商素材生成器 · 啟動說明

## 📋 使用前準備

### 1. 取得 Gemini API Key
前往 → [https://aistudio.google.com/apikey](https://aistudio.google.com/apikey)
點擊「Create API Key」，複製後備用。

> ⚠️ **圖片生成（Step 3）需要付費 API Key**，文字分析與文案生成（Step 2、4）可使用免費層。

---

## 🚀 本機啟動方式

### 方法一：直接執行（推薦）

```bash
# 1. 安裝套件
pip install -r requirements.txt

# 2. 啟動 App
streamlit run app.py
```

瀏覽器會自動開啟 → `http://localhost:8501`

---

### 方法二：使用虛擬環境（較乾淨）

```bash
# 建立虛擬環境
python -m venv venv

# 啟動虛擬環境
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 安裝套件
pip install -r requirements.txt

# 啟動 App
streamlit run app.py
```

## 💡 常見問題

**Q: Step 3 圖片生成失敗？**
A: 請確認 API Key 有 `gemini-3.1-flash-image-preview` 的使用配額。免費層目前不支援圖片生成，需升級至付費方案。

**Q: 如何修改提示詞風格？**
A: Step 2 完成後，所有提示詞都可直接在畫面上編輯，修改後直接點「生成」按鈕即可。

**Q: 可以換成其他電商商品嗎？**
A: 可以！直接上傳任何商品平拍照，Gemini 會自動分析並生成對應的模特兒場景提示詞。
